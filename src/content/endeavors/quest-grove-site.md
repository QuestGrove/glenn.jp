Quest Grove started as a frustration with every other form of online presence I had tried.

Social media felt like performing. A portfolio felt like a sales pitch. A blog felt like I needed to be an expert before I could say anything.

None of those felt like a place I would actually *return* to.

The idea of a base camp felt right. Somewhere functional, honest, and mine. Somewhere that gets better the more I use it, not the more I curate it.

## What I'm building

A static site using Astro and Obsidian. The technical choices were made to serve the writing, not the other way around. Markdown files I can edit anywhere. A site that builds fast and needs no server.

## What I'm learning along the way

How to structure content systems. How to think about information architecture. That learning to program at 35 with no background is genuinely difficult and genuinely rewarding at the same time.
